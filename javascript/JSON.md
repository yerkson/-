        * 	- JSON
        * 		- JavaScript Object Notation JS对象表示法
        * 		- JSON和JS对象的格式一样，只不过JSON字符串中的属性名必须加双引号
        * 			其他的和JS语法一致
        * 		JSON分类：
        * 			1.对象 {}
        * 			2.数组 []
        * 
        * 		JSON中允许的值：
        * 			1.字符串
        * 			2.数值
        * 			3.布尔值
        * 			4.null
        * 			5.对象
        * 			6.数组
        
        
     *  var json = '{"name":"孙悟空","age":18,"gender":"男"}';
     *  json --> js对象
     * 	 JSON.parse()
     * 		- 可以将以JSON字符串转换为js对象
     * 		- 它需要一个JSON字符串作为参数，会将该字符串转换为JS对象并返回


     * JS对象 ---> JSON
     * 	JSON.stringify()
     * 		- 可以将一个JS对象转换为JSON字符串
     * 		- 需要一个js对象作为参数，会返回一个JSON字符串
